# 問題分類

## 類型

**治療型 (Therapeutic)**

## 推理過程

1. **I 的性質分析**: 介入為 Dapagliflozin 10mg（SGLT2 抑制劑）— 這是一種藥物治療
2. **O 的性質分析**: 結果為腎功能惡化（eGFR 下降、末期腎病、腎臟相關死亡）— 這是臨床結果改善指標
3. **結論**: I 為藥物治療 + O 為臨床結果改善 → **治療型**

## 最佳證據層級

1. Systematic Review / Meta-analysis of RCTs
2. Individual RCT
3. Cohort study
4. Case-control study
5. Case series / Case report

## PubMed 搜尋 Filter

```
Randomized Controlled Trial[pt] OR Meta-Analysis[pt] OR Systematic Review[pt]
```
