# Style D — 競賽完整風格

最完整的 5A + GRADE + SDM + 成本效益，適合 EBM 競賽。

## 設計特徵
- **配色**: 雙色漸層（深藍到淺藍 / 深綠到淺綠），專業且吸睛
- **導航**: 側邊 5A 標籤 + 頂部進度條雙重導航
- **過場頁**: 全版面 5A 圖 + 該階段的預告重點
- **字體**: 標題用特色字體，內文標準
- **排版**: 資訊密度高但有清楚的視覺層次
- **特色**: 內容最完整，包含 GRADE、OCEBM、SDM、成本效益全套

## 投影片風格
- 封面：雙色漸層 + 大標題 + 副標題
- PICO：精美表格 + MeSH terms
- 搜尋：完整 PRISMA 流程圖 + 各資料庫截圖
- 評讀：RoB 2.0 交通燈圖 / AMSTAR 2 星級評等
- 結果：Forest Plot + 數據表格 + NNT 視覺化
- GRADE：證據品質評定表
- SDM：醫病共享決策流程圖
- 成本效益：表格化比較

## 完整段落配置
1. 封面 + 目錄（2 張）
2. ASK：臨床場景 + 背景 + PICO + 問題分類（10-12 張）
3. ACQUIRE：6S 搜尋 + PRISMA + 選文（12-16 張）
4. APPRAISE：評讀 + GRADE + 結果（18-25 張）
5. APPLY：OCEBM + 臨床應用 + SDM + 成本效益 + 去學術化回覆（8-10 張）
6. AUDIT：五面向自我評估（4-5 張）
7. References + 致謝（2 張）
**總計：55-70 張**

- 全國 EBM 競賽
