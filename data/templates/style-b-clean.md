# Style B — 簡潔現代風格

大量留白 + 圖示化，適合視覺化呈現。

## 設計特徵
- **配色**: 白底 + 單一主色（可選藍/綠/橘），極簡風
- **導航**: 頂部進度條（5A 五格，填色標記進度）
- **過場頁**: 全版面單一大字（如「APPRAISE」）+ 副標題
- **字體**: 無襯線體，標題特大，內文精簡
- **排版**: 每頁一個重點，大量使用 icon 和圖示
- **特色**: 視覺清爽、適合投影、觀眾容易跟隨

## 投影片風格
- 封面：大圖背景 + 標題
- 內容頁：左圖右文 或 上圖下文
- PICO：四格圖示卡片
- 搜尋策略：流程圖視覺化
- 評讀：圖表化（紅綠燈 / 儀表板）
- Forest Plot：直接嵌入圖片
