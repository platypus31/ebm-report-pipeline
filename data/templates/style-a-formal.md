# Style A — 正式學術風格

側邊欄導航 + 5A 標籤標記，適合正式 EBM 發表場合。

## 設計特徵
- **配色**: 深藍 / 白底，專業學術感
- **導航**: 每張投影片側邊欄顯示 5A 步驟標籤（問題 ASK / 檢索 ACQUIRE / 評讀 APPRAISAL / 應用 APPLY / 評估 AUDIT），高亮當前步驟
- **過場頁**: 5A 流程總覽圖，高亮當前階段
- **字體**: 標題用粗體，內文用標準字
- **排版**: 條列式為主，每頁 3-5 點
- **特色**: 嚴謹、結構化、適合主治醫師審閱

## 投影片風格
- 封面：大標題居中，科別 + 日期 + 報告者
- 內容頁：左側導航欄（5A 標籤） + 右側主內容
- 表格：藍白交替色
- 圖表：簡潔線條
- 評讀結果：綠色（Yes）/ 紅色（No）/ 灰色（Can't tell）

## 適合
- 科內 EBM 正式報告
- 院內 EBM 競賽
- 教學醫院評鑑
