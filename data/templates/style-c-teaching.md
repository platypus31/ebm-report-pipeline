# Style C — 教學導向風格

6A 擴充框架 + 詳細步驟解說，適合 EBM 初學者。

## 設計特徵
- **配色**: 暖色調（淺黃 / 橘 / 白），親切感
- **導航**: 6A 流程圖（Analysis + Ask + Acquire + Appraise + Apply + Audit），每個階段有小 icon
- **過場頁**: 6A 流程圖 + 當前步驟的教學提示
- **字體**: 較大字體，適合遠距離觀看
- **排版**: 左右對比式（做法 vs 原文佐證）
- **特色**: 每個步驟附帶教學說明，幫助觀眾學習 EBM 方法

## 投影片風格
- 封面：溫馨設計，標題 + 小圖
- PICO：逐步拆解動畫效果
- 搜尋：每個資料庫附上截圖位置說明
- 評讀：每題附 checklist 教學提示（「為什麼這題重要」）
- 自我評估：互動式問答設計

## 特殊元素
- 每個 5A/6A 階段前有「這個階段要做什麼」的教學頁
- 評讀問題附上「判斷要點」提示框
- 臨床回覆附上「去學術化範例」對照

## 適合
- PGY 第一次做 EBM 報告
- EBM 工作坊
- 跨科教學
